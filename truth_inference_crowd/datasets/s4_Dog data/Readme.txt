The dog data contains 4 classes denoted by 0-3. You can see a detailed description of the dataset in our NIPS 12 paper (section 6.2): 

D. Zhou, J. C. Platt, S. Basu, and Y. Mao. Learning from the Wisdom of Crowds by Minimax Entropy. Advances in Neural Information Processing Systems (NIPS) 25, 2204-2212, 2012.   

The file in “dogs-1.merged.label.tsv” contains 3 columns:  item ID, worker ID, worker label 

The file in “dogs-1.merged.truth.tsv” contains 2 columns:  item ID, true label 

Run your algorithm with the label file and evaluate the results with the truth file.
